Half 2 — R ≤ 3 If R were > 3, the series (= f wherever it converges) would have to stay bounded on a disk reaching past z = 3. It doesn't: |f(z)| has no finite ceiling as z → 3. So R can't exceed 3. General relation for real z < 3: |f(z)| = 1/|z−3| = 1/(3−z). If R > 3, f would stay bounded past z = 3. It doesn't — |f(z)| → ∞. So R ≤ 3.
